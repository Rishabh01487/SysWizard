# SysWizard

Interactive system design learning platform with animated visualizations for 64+ topics.

## Features
- 🎨 64 animated system design topic visualizations
- 🧙‍♂️ AI-powered tutoring (Ollama)
- 🔐 User authentication with progress tracking
- 📊 SVG architecture diagrams for every topic
- 🎯 Quizzes, algorithms, code examples, interview tips
- 🎬 Animation recording

## Tech Stack
- Vite + Vanilla JS
- Canvas-based animation engine
- localStorage-based auth

## Setup
```bash
npm install
npm run dev
```
